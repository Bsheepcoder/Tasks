# 2.CSS                                                                                                                                                                                                                                                                                                                                                                      

CSS 指层叠样式表 (**C**ascading **S**tyle **S**heets)，分为内部样式和外部样式

==内部样式表==是定义在head元素标签内的一种“特征”定义，和html的内部样式相同，形式类似函数

==外部样式表==是定义在外部的CSS文件中，作用与内部样式相同

==内联样式表==是定义在标签元素内部的样式表

***

## id和class选择器
